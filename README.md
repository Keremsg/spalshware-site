# 🚀 Setup

```bash
git clone https://github.com/keremsg/spalshware-site.git
cd spalshware-site
npm install
```

Create a `.env` file:

```env
TOKEN=YOUR_BOT_TOKEN
CLIENT_ID=YOUR_CLIENT_ID
```

Start the bot:

```bash
node .
```
