# 🚀 Setup

```bash
git clone https://github.com/Keremsg/spalshware-site.git

cd spalshware-site

npm install
```

Create `.env`

```env
TOKEN=YOUR_TOKEN
CLIENT_ID=YOUR_CLIENT_ID
```

Start Bot

```bash
node .
```
